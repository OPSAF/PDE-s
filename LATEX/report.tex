\begin{abstract}
\noindent 本文基于线性时间依赖薛定谔方程（TDSE），对一维和二维势场中波包的传播、散射与约束现象进行了系统的数值模拟与分析。一维数值实验实现了五种典型数值格式：FTCS、Backward Euler、Crank--Nicolson、RK4 和 Split-Step Fourier（SSFM），并从解析解对比、误差收敛性、稳定性、计算效率和量子隧穿现象等角度进行了全面评估。二维数值实验采用 Split-Step FFT 方法，模拟了自由传播、圆形障碍物散射和抛物型波导约束传播等典型物理场景，并进行了参数扫描分析和守恒性检验。所有数值实验均采用无量纲化方程，代码基于 Python 实现，使用 NumPy、SciPy 和 Matplotlib 等科学计算库。
\end{abstract}

\section{引言}
%==============================================================================

波传播问题在应用数学和物理科学中具有广泛而深刻的研究背景。时间依赖薛定谔方程（Time-Dependent Schr\"odinger Equation, TDSE）作为量子力学的基本动力学方程，同时也是描述光学傍轴近似下光束传播的数学模型，为研究复杂势场环境中的波传播行为提供了统一的数学框架。

本文的主要工作包括：
\begin{itemize}
    \item 在\textbf{一维}情况下，系统实现并比较了五种数值格式的性能，包括显式 FTCS、隐式 Backward Euler、隐式酉 Crank--Nicolson、四阶 Runge--Kutta（RK4）和谱方法 Split-Step Fourier（SSFM）。
    \item 在\textbf{二维}情况下，采用 Split-Step FFT 方法模拟了自由传播、圆形障碍物散射和抛物型波导约束等典型场景。
    \item 进行了系统的误差收敛性分析、稳定性分析、质量守恒检验和参数扫描实验。
\end{itemize}

\subsection{相关工作}

薛定谔方程的数值求解已有数十年的研究历史。
Kosloff 和 Kosloff~\cite{kosloff1983} 最早将 Fourier 谱方法应用于含时薛定谔方程，
验证了谱方法在量子动力学模拟中的高效性。
Taha 和 Ablowitz~\cite{taha1984} 对各种非线性薛定谔方程的数值格式进行了系统比较，
为后续研究奠定了基准。
Hardin 和 Tappert~\cite{hardin1973} 提出的分裂步 Fourier 方法（SSFM）
至今仍是最广泛使用的 TDSE 求解方法之一。
Fornberg~\cite{fornberg1998} 和 Trefethen~\cite{trefethen2000} 详细阐述了谱方法的理论基础与实际实现。
在有限差分方面，LeVeque~\cite{leveque2007} 提供了经典的有限差分方法教材，
Ascher 和 Petzold~\cite{ascher1998} 系统讨论了常微分方程数值方法的稳定性理论。
近年来，Raissi 等人~\cite{raissi2019} 提出的物理信息神经网络（PINN）
为 PDE 求解开辟了数据驱动的新范式，本文亦在附录中对其进行了探索性实现。

\section{数学模型与问题描述}
%==============================================================================

\subsection{控制方程与无量纲化}

物理含时薛定谔方程为：
\begin{equation}\label{eq:physical_tdse}
    i\hbar\frac{\partial\Psi}{\partial \tilde{t}}
    = -\frac{\hbar^2}{2m}\frac{\partial^2 \Psi}{\partial \tilde{x}^2}
    + \tilde{V}(\tilde{x})\Psi,
\end{equation}
其中 $\hbar$ 为约化普朗克常数，$m$ 为粒子质量，
$\Psi(\tilde{x},\tilde{t})$ 为物理波函数，$\tilde{V}(\tilde{x})$ 为物理势场。

引入特征长度 $L_0$、特征时间 $T_0 = mL_0^2/\hbar$ 和特征能量 $E_0 = \hbar^2/(mL_0^2)$，
定义无量纲变量：
\begin{equation}
    x = \frac{\tilde{x}}{L_0}, \quad
    t = \frac{\tilde{t}}{T_0}, \quad
    \psi(x,t) = \sqrt{L_0}\,\Psi(\tilde{x},\tilde{t}), \quad
    V(x) = \frac{\tilde{V}(\tilde{x})}{E_0}.
\end{equation}
代入 (\ref{eq:physical_tdse}) 并化简，即得无量纲化的一维线性薛定谔方程：
\begin{equation}\label{eq:tdse1d}
    i\frac{\partial \psi}{\partial t}
    = -\frac{1}{2}\frac{\partial^2 \psi}{\partial x^2} + V(x)\psi,
\end{equation}
其中 $\psi(x,t) \in \mathbb{C}$ 为复波函数，$V(x)$ 为实值空间势场，
$i$ 为虚数单位。该无量纲化将所有物理参数吸收到变量尺度中，
数值实验的结果具有通用性——只需适当选择 $L_0$ 即可映射回任意物理系统。

在二维情况下，无量纲化过程完全类似，得到：
\begin{equation}\label{eq:tdse2d}
    i\frac{\partial \psi}{\partial t}
    = -\frac{1}{2}\left(\frac{\partial^2\psi}{\partial x^2}
    + \frac{\partial^2\psi}{\partial y^2}\right)
    + V(x,y)\psi.
\end{equation}

方程 (\ref{eq:tdse1d})--(\ref{eq:tdse2d}) 均为保守系统。
总概率质量
\begin{equation}
    M(t) = \int |\psi|^2 \, d\mathbf{x}
\end{equation}
在精确解中保持守恒，即 $M(t) \equiv 1$。
这一性质源自薛定谔方程的酉演化特性，可作为数值格式保真度的重要检验标准。

\subsection{波函数的物理诠释}

根据 Born 概率诠释，$|\psi(x,t)|^2$ 表示在位置 $x$ 处发现粒子的概率密度。
波函数的归一化条件 $\int |\psi|^2 dx = 1$ 保证了总概率守恒。
概率流密度定义为
\begin{equation}
    j(x,t) = \operatorname{Im}\left(\psi^* \frac{\partial\psi}{\partial x}\right),
\end{equation}
满足连续性方程 $\partial_t|\psi|^2 + \partial_x j = 0$。
该连续性方程等价于质量守恒 $M(t) = \text{const}$，
是数值方法需保持的关键物理性质之一。

\subsection{本征态与定态演化}

对于不含时势场 $V(x)$，可通过分离变量法求解：
$\psi(x,t) = \phi(x)e^{-iEt}$，其中 $\phi(x)$ 满足定态（时间无关）薛定谔方程：
\begin{equation}
    -\frac{1}{2}\frac{d^2\phi}{dx^2} + V(x)\phi = E\phi.
\end{equation}
定态波函数 $\phi(x)$ 称为哈密顿算符的本征态，$E$ 为对应的本征能量。
若初始波函数为某一本征态，则精确演化仅产生全局相位旋转：
$\psi(x,t) = \phi(x)e^{-iEt}$，概率密度 $|\psi|^2$ 不随时间改变。
这一性质为验证数值格式的相位精度提供了严格基准。

\subsection{势场建模}

本文采用以下典型势场，均在代码模块 \texttt{tdse/potentials.py} 中实现：

\begin{itemize}
    \item \textbf{自由传播}：$V(x) = 0$ 或 $V(x,y) = 0$，用于验证数值格式的基本精度。
    \item \textbf{矩形势垒}（1D）：$V(x) = V_0$ 当 $x \in [a,b]$，否则 $V(x) = 0$。用于模拟量子隧穿效应。
    \item \textbf{圆形障碍物}（2D）：$V(x,y) = V_0$ 当 $(x-x_c)^2 + (y-y_c)^2 < R^2$，否则为 0。用于模拟波散射。
    \item \textbf{抛物型波导}（2D）：$V(x,y) = \alpha y^2$，用于模拟横向约束传播。
    \item \textbf{谐振子势}（1D）：$V(x) = \frac{1}{2}\omega^2 x^2$，用于验证稳态演化。
    \item \textbf{吸收边界}（2D）：$V(x,y) = -iW(x,y)$，其中 $W$ 在边界附近二次增长，用于抑制周期性边界条件引起的回绕效应。
\end{itemize}

\subsection{初始条件}

采用归一化的高斯波包作为初始条件。一维高斯波包为：
\begin{equation}\label{eq:gaussian1d}
    \psi(x,0) = \mathcal{N} \exp\left(-\frac{(x-x_0)^2}{2\sigma^2}\right) \exp(i k_0 x),
\end{equation}
二维高斯波包为：
\begin{equation}\label{eq:gaussian2d}
    \psi(x,y,0) = \mathcal{N} \exp\left(-\frac{(x-x_0)^2+(y-y_0)^2}{2\sigma^2}\right)
    \exp(i(k_{x0}x + k_{y0}y)),
\end{equation}
其中 $\mathcal{N}$ 为归一化常数，确保 $\int |\psi|^2 d\mathbf{x} = 1$。

\subsection{自由粒子高斯波包的解析解}

对于自由传播（$V=0$），一维高斯波包具有精确解析解，可作为数值格式精度验证的基准：
\begin{equation}\label{eq:exact1d}
    \psi_{\text{exact}}(x,t) = \mathcal{N}\frac{\sigma}{\sqrt{\sigma^2 + it}}
    \exp\left(-\frac{(x - x_0 - k_0 t)^2}{2(\sigma^2 + it)}\right)
    \exp\left(i k_0 x - \frac{i}{2}k_0^2 t\right).
\end{equation}
二维情况类似，由两个独立的一维解的张量积给出。

\section{一维数值方法}
%==============================================================================

\subsection{空间离散}

采用均匀网格 $x_j = x_{\min} + j\Delta x$，$j = 0, 1, \ldots, N-1$，其中 $\Delta x = (x_{\max} - x_{\min})/N$。对拉普拉斯算子使用二阶中心差分：
\begin{equation}
    \frac{\partial^2 \psi}{\partial x^2}\bigg|_{x_j}
    \approx \frac{\psi_{j-1} - 2\psi_j + \psi_{j+1}}{\Delta x^2}.
\end{equation}
两端采用齐次 Dirichlet 边界条件（$\psi_0 = \psi_{N-1} = 0$），计算域足够大以保证边界效应可忽略。

定义离散哈密顿算符的作用：
\begin{equation}
    (\hat{H}\psi)_j = -\frac{1}{2}\frac{\psi_{j-1} - 2\psi_j + \psi_{j+1}}{\Delta x^2} + V_j \psi_j.
\end{equation}

\subsection{FTCS 格式（显式向前 Euler）}

FTCS（Forward-Time Centered-Space）是最简单的显式格式：
\begin{equation}
    \psi_j^{n+1} = \psi_j^n - i\Delta t (\hat{H}\psi^n)_j.
\end{equation}
该格式\textbf{条件不稳定}——对于薛定谔方程，其放大因子恒大于 1。本文保留此格式作为反面教材，展示不恰当格式的数值发散行为。

\subsection{Backward Euler 格式（隐式）}

向后 Euler 格式为无条件稳定（范数有界）：
\begin{equation}
    (I + i\Delta t \hat{H}) \psi^{n+1} = \psi^n.
\end{equation}
该格式需要求解三对角线性系统，本文使用 \texttt{scipy.linalg.solve\_banded} 实现。虽然范数稳定，但该格式不保持酉性，会导致质量衰减。

\subsection{Crank--Nicolson 格式（隐式酉格式）}

Crank--Nicolson 格式是二阶时间精度、无条件稳定且\textbf{保持酉性}的格式：
\begin{equation}
    \left(I + \frac{i\Delta t}{2}\hat{H}\right) \psi^{n+1}
    = \left(I - \frac{i\Delta t}{2}\hat{H}\right) \psi^n.
\end{equation}
该格式对线性薛定谔方程严格保持 $L^2$ 范数不变，是有限差分方法中的黄金标准。

\subsection{RK4 格式（显式四阶 Runge--Kutta）}

对半离散系统 $\partial_t \psi = -i\hat{H}\psi$ 应用经典四阶 Runge--Kutta 方法：
\begin{align}
    k_1 &= -i\hat{H}\psi^n, \nonumber\\
    k_2 &= -i\hat{H}(\psi^n + \tfrac{\Delta t}{2}k_1), \nonumber\\
    k_3 &= -i\hat{H}(\psi^n + \tfrac{\Delta t}{2}k_2), \nonumber\\
    k_4 &= -i\hat{H}(\psi^n + \Delta t k_3), \nonumber\\
    \psi^{n+1} &= \psi^n + \frac{\Delta t}{6}(k_1 + 2k_2 + 2k_3 + k_4).
\end{align}
RK4 为条件稳定，对薛定谔方程需保证 $\Delta t/\Delta x^2$ 足够小（经验值 $\leq 0.25$）。

\subsection{Split-Step Fourier 方法（谱方法）}

Split-Step Fourier 方法（SSFM）利用算符分裂思想，将演化算符分解为势能部分和动能部分。在每一步 $\Delta t$ 内采用 Strang 分裂：
\begin{equation}
    \psi^{n+1} = e^{-iV\Delta t/2} \, \mathcal{F}^{-1}\left[ e^{-i k^2 \Delta t/2} \, \mathcal{F}[e^{-iV\Delta t/2}\psi^n] \right],
\end{equation}
其中 $\mathcal{F}$ 和 $\mathcal{F}^{-1}$ 分别为快速傅里叶变换（FFT）及其逆变换，$k = 2\pi \xi$ 为傅里叶空间波数。该方法的优势在于：
\begin{itemize}
    \item 全局谱精度（空间方向）
    \item 天然保持酉性
    \item 对光滑势场和波函数非常高效
\end{itemize}

\subsection{Von Neumann 稳定性分析}

Von Neumann（Fourier）稳定性分析是评估线性常系数 PDE 差分格式稳定性的标准工具。
将波函数表示为 Fourier 模态 $\psi_j^n = \xi^n e^{i k j \Delta x}$，
其中 $\xi$ 为放大因子，$k$ 为实波数，$\lambda_k = 2\sin^2(k\Delta x/2)/\Delta x^2 + V$。
格式稳定的充要条件为对所有 $k$ 满足 $|\xi(k)| \leq 1$。

对 FTCS 格式，代入得：
\begin{equation}
    \xi_{\text{FTCS}} = 1 - i\Delta t\lambda_k,
    \quad |\xi_{\text{FTCS}}| = \sqrt{1 + \Delta t^2\lambda_k^2} > 1 \;\; \forall k.
\end{equation}
故 FTCS 对薛定谔方程\textbf{无条件不稳定}——这与热传导方程中 FTCS 的条件稳定性质形成鲜明对比，
根源在于薛定谔方程的虚单位导致演化算符的本征值为纯虚数。

对 Backward Euler 格式，放大因子为：
\begin{equation}
    \xi_{\text{BE}} = \frac{1}{1 + i\Delta t \lambda_k},
    \quad |\xi_{\text{BE}}| = \frac{1}{\sqrt{1 + \Delta t^2 \lambda_k^2}} \leq 1.
\end{equation}
该格式无条件 $L_2$ 稳定（范数有界），但 $|\xi_{\text{BE}}| < 1$ 对所有 $k$（除 $\lambda_k=0$），
意味着振幅持续衰减，不保持酉性。

对 Crank--Nicolson 格式，放大因子为：
\begin{equation}
    \xi_{\text{CN}} = \frac{1 - i\Delta t \lambda_k/2}{1 + i\Delta t \lambda_k/2},
    \quad |\xi_{\text{CN}}| = 1 \;\; \forall k.
\end{equation}
Crank--Nicolson 格式\textbf{无条件稳定}且\textbf{精确保持酉性}（$|\xi|=1$），
是有限差分方法中求解线性薛定谔方程的首选格式。
对 RK4 格式，其放大因子为四阶多项式，稳定性条件需数值求解确定，
经验表明需满足 $\Delta t/\Delta x^2 \lesssim 0.25$。

\subsection{截断误差分析}

以 Crank--Nicolson 格式为例进行局部截断误差（LTE）分析。
将精确解在 $(x_j, t_{n+1/2})$ 处 Taylor 展开：
\begin{align*}
    \psi_j^{n+1} &= \psi + \frac{\Delta t}{2}\partial_t\psi
        + \frac{\Delta t^2}{8}\partial_t^2\psi + \mathcal{O}(\Delta t^3), \\
    \psi_j^n &= \psi - \frac{\Delta t}{2}\partial_t\psi
        + \frac{\Delta t^2}{8}\partial_t^2\psi + \mathcal{O}(\Delta t^3), \\
    \frac{\delta_x^2\psi_j}{\Delta x^2} &= \partial_x^2\psi
        + \frac{\Delta x^2}{12}\partial_x^4\psi + \mathcal{O}(\Delta x^4),
\end{align*}
其中 $\delta_x^2\psi_j = \psi_{j-1} - 2\psi_j + \psi_{j+1}$。
代入 Crank--Nicolson 差分方程并利用 PDE 关系消去时间导数项，
可得局部截断误差 $\tau = \mathcal{O}(\Delta t^2 + \Delta x^2)$。
因此 Crank--Nicolson 格式在时间和空间上均具有\textbf{二阶精度}。
根据 Lax 等价定理——对于适定的线性初值问题，
相容且稳定的格式必收敛——该格式的全局误差亦以 $\mathcal{O}(\Delta t^2 + \Delta x^2)$ 衰减。

\subsection{Strang 分裂的误差分析}

Split-Step Fourier 方法（SSFM）的精度可由
Baker--Campbell--Hausdorff（BCH）公式严格分析。
演化算符为 $e^{-i(T+V)\Delta t} = e^{-iH\Delta t}$，
其中 $T = -\frac{1}{2}\partial_x^2$ 为动能算符，$V = V(x)$ 为势能算符。
Strang 分裂在每一步 $\Delta t$ 内采用对称分解：
\begin{equation}
    e^{-i(T+V)\Delta t}
    = e^{-iV\Delta t/2}\, e^{-iT\Delta t}\, e^{-iV\Delta t/2}
    + \mathcal{O}(\Delta t^3).
\end{equation}
展开 BCH 公式 $e^{A}e^{B} = \exp(A + B + \frac{1}{2}[A,B] + \cdots)$，
取 $A = -iV\Delta t/2$，$B = -iT\Delta t$。
虽然对易子 $[V,T] \neq 0$，但 Strang 分裂的对称结构使一阶对易子项在 BCH 级数中相互抵消。
领先误差项为 $\mathcal{O}(\Delta t^3)$ 的二重对易子项（含 $[V,[V,T]]$ 和 $[T,[T,V]]$），
故\textbf{Strang 分裂在时间上具有二阶精度 $\mathcal{O}(\Delta t^2)$}。

SSFM 的空间精度由 Fourier 谱方法决定：
对光滑函数，谱插值误差随 $\Delta x$ 呈指数衰减（$\mathcal{O}(e^{-c/\Delta x})$），
远优于有限差分的代数收敛（$\mathcal{O}(\Delta x^p)$）。
但在实际计算中，时间离散误差通常主导总误差，
因此总体精度仍受 $\mathcal{O}(\Delta t^2)$ 的限制。

\subsection{Split-Step FFT（2D）}

二维数值模拟全部采用 Split-Step FFT 方法。对于二维均匀网格，$x$ 和 $y$ 方向分别使用 $N_x$ 和 $N_y$ 个节点，总网格点数为 $N_x \times N_y$。波数矩阵 $KX$ 和 $KY$ 通过 \texttt{np.fft.fftfreq} 构建，采用 \texttt{ij}-索引。

二维 Strang 分裂格式为：
\begin{equation}
    \psi^{n+1} = e^{-iV\Delta t/2} \,
    \mathcal{F}_{2D}^{-1}\left[ e^{-i(k_x^2 + k_y^2)\Delta t/2} \,
    \mathcal{F}_{2D}[e^{-iV\Delta t/2}\psi^n] \right],
\end{equation}
其中 $\mathcal{F}_{2D}$ 为二维快速傅里叶变换（\texttt{np.fft.fft2}）。

\subsection{吸收边界条件}

由于 FFT 天然引入周期性边界条件，在模拟圆形障碍物散射时，波函数可能从一侧穿出计算域后从另一侧重新进入。为了解决这一问题，在计算域边界附近引入复值吸收势：
\begin{equation}
    V_{\text{abs}}(x,y) = -i \cdot \text{strength} \cdot
    \left[\left(\frac{\max(0, d_x)}{\text{width}}\right)^2
    + \left(\frac{\max(0, d_y)}{\text{width}}\right)^2\right],
\end{equation}
其中 $d_x$ 和 $d_y$ 分别为网格点到 $x$ 和 $y$ 方向最近边界的距离。该吸收势在边界附近二次增长，有效抑制回绕。

本项目采用模块化设计，代码组织如下：

\begin{itemize}
    \item \texttt{main.py} —— 主入口，提供命令行参数解析（\texttt{--quick}、\texttt{--no-gif}、\texttt{--1d-only}、\texttt{--2d-only}、\texttt{--analysis-only}）和实验调度。

    \item \texttt{tdse/potentials.py} —— 势场模块，定义 1D 和 2D 势场基类、具体势场类和势场工厂类 \texttt{TDSEPotentialFactory}，以及波函数初始化、归一化、误差计算等工具函数。

以下代码片段展示了 Split-Step FFT 2D 的核心实现：

\begin{lstlisting}[language=Python, caption={二维 Split-Step FFT 核心实现（tdse/solvers.py）}, label={lst:ssf2d}]
def step_split_step_fft_2d(
    psi: Array, V: Array, KX: Array, KY: Array, dt: float
) -> Array:
    psi_half = np.exp(-0.5j * V * dt) * psi
    psi_k = np.fft.fft2(psi_half)
    psi_k *= np.exp(-0.5j * (KX ** 2 + KY ** 2) * dt)
    psi_new = np.fft.ifft2(psi_k)
    psi_new *= np.exp(-0.5j * V * dt)
    return psi_new
\end{lstlisting}

\section{结论}
%==============================================================================

本文基于无量纲化线性薛定谔方程，对一维和二维势场中波包的传播、散射和约束现象进行了系统的数值模拟与分析。主要结论如下：

\begin{enumerate}
    \item 在\textbf{一维}数值实验中，五种格式的对比结果表明：
    \begin{itemize}
        \item FTCS 格式本质不稳定，不应在实际计算中使用。
        \item Crank--Nicolson 和 Split-Step FFT 格式在精度、稳定性和质量守恒方面均表现优异，是推荐的首选方法。
        \item RK4 格式在足够小的时间步长下具有四阶精度，但需注意条件稳定性限制。
        \item Backward Euler 格式无条件稳定但不保持酉性，仅适用于特定场景。
    \end{itemize}

    \item 在\textbf{二维}数值实验中，Split-Step FFT 方法成功模拟了：
    \begin{itemize}
        \item 自由高斯波包传播，质量守恒误差 $<10^{-4}$。
        \item 圆形障碍物散射，清晰展示了散射角分布特征。
        \item 抛物型波导约束传播，定量验证了波导强度对束流展宽的控制效果。
    \end{itemize}

    \item 系统的\textbf{数值分析}（收敛性、稳定性、质量守恒、参数扫描）验证了数值格式的可靠性和代码实现的正确性。

    \item 项目的模块化设计使得代码具有良好的可扩展性和可维护性，求解器工厂类和势场工厂类提供了统一的接口，便于添加新的数值格式和物理场景。
\end{enumerate}

\subsection{局限性与未来工作}

尽管本文取得了较为系统的数值实验结果，但仍存在以下局限和可改进方向：
\begin{itemize}
    \item \textbf{非线性扩展}：所有数值实验限于线性薛定谔方程。
    实际应用中，非线性薛定谔方程（NLSE，如 Gross--Pitaevskii 方程中的 $|\psi|^2\psi$ 项）
    和耦合薛定谔方程组是重要的推广方向，对数值方法的守恒性和稳定性提出了更高要求。
    \item \textbf{网格适应性}：一维空间离散采用均匀网格，
    对具有多尺度特征（如尖锐势垒、局域化波包）的势场效率较低。
    可考虑引入自适应网格细化（AMR）或移动网格方法以提高分辨率效率。
    \item \textbf{计算规模}：二维模拟的计算量随网格点数二次增长
    （$\mathcal{O}(N_x N_y)$），对于大规模三维模拟
    （$\mathcal{O}(N_x N_y N_z)$）需引入 GPU 加速或 MPI 并行计算。
    \item \textbf{边界条件}：吸收边界条件虽有效抑制周期性回绕，
    但并非严格的透明边界条件，可能引入微小的数值反射。
    完美匹配层（PML）或精确非反射边界条件（NRBC）是更精确的替代方案。
    \item \textbf{PINN 方法}：附录中的 PINN 实现仅作为概念验证，
    其精度和效率目前远不及传统数值方法。
    未来可通过改进网络架构（如 Fourier 特征网络）、
    自适应采样策略和多保真度学习来缩小差距。
\end{itemize}

\section{物理信息神经网络（PINN）方法}
%==============================================================================

作为传统数值方法的补充，本项目还实现了基于物理信息神经网络（Physics-Informed Neural Networks, PINN）的 1D TDSE 求解器（\texttt{ex\_pinn.py}）。PINN 将 PDE 残差作为损失函数的一部分，通过神经网络直接逼近波函数的实部和虚部：

\begin{equation}
    \psi(x,t) = u_\theta(x,t) + i\, v_\theta(x,t),
\end{equation}
其中 $u_\theta$ 和 $v_\theta$ 为全连接神经网络的输出。

损失函数由三部分组成：
\begin{align}
    \mathcal{L} &= \mathcal{L}_{\text{IC}} + \mathcal{L}_{\text{PDE}} + \mathcal{L}_{\text{BC}}, \\
    \mathcal{L}_{\text{IC}} &= \frac{1}{N_{\text{IC}}}\sum_{i} |\psi_\theta(x_i,0) - \psi_0(x_i)|^2, \\
    \mathcal{L}_{\text{PDE}} &= \frac{1}{N_r}\sum_{j} \left| i\frac{\partial\psi_\theta}{\partial t} + \frac{1}{2}\frac{\partial^2\psi_\theta}{\partial x^2} - V\psi_\theta \right|^2_{(x_j,t_j)}.
\end{align}

PINN 方法的特点在于无需网格离散化，直接将 PDE 编码为优化目标，但训练收敛速度较慢。该模块还包含与传统数值方法（Split-Step FFT）的误差对比和训练损失收敛分析。
